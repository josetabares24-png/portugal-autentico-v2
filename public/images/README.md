## Para la Previsualización

Las imágenes aparecerán con errores de carga, pero esto no afecta la funcionalidad del diseño responsive ni la estructura de la página. El diseño, colores, tipografía, navegación y layout funcionarán perfectamente.

## Imágenes Recomendadas

Para la versión final, se recomienda usar:
- Fotos profesionales de alta calidad
- Formato JPG o WebP
- Resolución mínima 1920px de ancho
- Imágenes de Unsplash o fotografías propias con licencia

### Fuentes Sugeridas
- Unsplash.com (búsqueda: "Lisboa", "Portugal", "Alfama")
- Pexels.com
- Fotografía propia en ubicaciones reales
